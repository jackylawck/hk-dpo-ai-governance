# Security Policy (安全性政策)

## Supported Versions (支援版本)
Only the latest release running on the [Live Sandbox App](https://hk-dpo-ai-governance.streamlit.app/) is actively supported.
本系統僅對部署於 [線上動態合規風險評分卡系統](https://hk-dpo-ai-governance.streamlit.app/) 的最新版本提供維護支援。

## Reporting a Vulnerability & Responsible Disclosure (報告漏洞與責任披露)
Please do not open public GitHub issues for security-related bugs or algorithmic vulnerabilities. 
若您發現系統存在安全性漏洞、數據洩露風險或演算法偏誤，請勿開設公開之 GitHub Issue。

To ensure a responsible disclosure process, please report any findings directly to the repository owner via LinkedIn:
為維護負責任的漏洞披露程序，請直接透過 LinkedIn 私下聯繫專案擁有者進行通報：
👉 **[Connect via LinkedIn (聯絡專案擁有者)](https://www.linkedin.com/in/jackylawck)**

We will review your report and mitigate verified compliance gaps promptly.
我們將第一時間覆核您的報告，並儘速修補經證實的合規缺口。
